# **<center>教學流程**

本教學提供從無到有建立論壇的方法
<ol>
  <li>首先需要有一個機器用來架設伺服器才能開始建立論壇（一、虛擬機建置）
  <li>當有了伺服器，接著就是在上面開始安裝discuz論壇（二、安裝）
  <li>接著是為了避免當伺服器發生預期外的問題或者檔案遺失的預防措施（三、備份）
  <li>最後是當上一點提到的問題真的發生且檔案遺失還原資料的方式（四、還原）
</ol>
